# Simple Web Backend App

A simple backend application built with **Spring Boot** and **H2 in-memory database**, demonstrating RESTful APIs and basic CRUD operations.

---

## Features

- RESTful APIs for CRUD operations
- H2 in-memory database for data persistence
- Built using Spring Boot and Maven
- Easy to run and extend

---

## Technologies Used

- Java 17+
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven
- IntelliJ IDEA (recommended)
